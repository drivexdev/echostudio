Common Questions

Got questions?

The answers to what most people ask before getting started.

How do I get a quote?
Reach out on Discord — it's the fastest way. Describe what you're building and we'll get back to you with a breakdown, usually within a few hours.
Do you work with existing servers?
Yes. Whether you need a plugin dropped into an existing setup or a full audit and rebuild, we handle both. We'll review what you have and advise on the best path forward.
What Minecraft versions do you support?
We work with the latest stable release (currently 1.21.x) and can support older versions depending on the project. Just mention your target version when you reach out.
Will my plugins work after MC updates?
We write clean, maintainable code specifically to make updates straightforward. We also offer ongoing maintenance packages if you want us to handle version bumps for you.
How long does a typical project take?
It depends on scope. A simple plugin can be done in a few days; a full server setup with custom plugins typically takes one to three weeks. We give you a realistic timeline before any work begins.
